<h2>📌 Sobre o projeto</h2>

<p>
  Este repositório contém o desenvolvimento do site institucional da
  <strong>AirFlow Films</strong>, criado para apresentar a marca, exibir o portfólio
  de vídeos e facilitar o contato com clientes interessados em produções audiovisuais
  profissionais.
</p>

<p>
  O projeto busca transmitir uma identidade visual moderna, minimalista e cinematográfica,
  reforçando a proposta premium da marca.
</p>

<hr>

<h2>🚀 Funcionalidades</h2>

<ul>
  <li>Interface moderna e responsiva</li>
  <li>Portfólio de vídeos</li>
  <li>Animações e transições suaves</li>
  <li>Botão de contato via WhatsApp</li>
  <li>Estrutura preparada para futuras integrações com backend</li>
</ul>
